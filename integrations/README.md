# Integrations

Appsmith integrates with multiple databases allowing you bring query & update your database directly.

### Supported Database Integrations:

* [Postgres](./postgres.md)
* MySQL
* [Mongo DB](./mongodb/README.md)
* Microsoft SQL Server
* Redis
* Elasticsearch
* Firestore
